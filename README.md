## 📂 Project Structure :

- `task-1-presentation` → Infographic on Digital Literacy  
- `task-2-portfolio` → Screenshots of GitHub, LinkedIn, Kaggle profiles  
- `task-3-platforms` → HackerRank solution + Google Form quiz
